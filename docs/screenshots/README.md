# Screenshots Guide

## Required Screenshots

Please capture the following screens and save them with these exact names:

1. **welcome-screen.png** - Onboarding/welcome page
2. **sign-in.png** - Login page
3. **register-dog.png** - Dog registration form (any step)
4. **match-swipe.png** - Main swipe interface with dog card
5. **match-preferences.png** - Filter preferences page
6. **profile.png** - User profile page

## How to Capture

### Option 1: Browser DevTools (Recommended)

1. Open app in browser: http://localhost:4200
2. Open DevTools (F12)
3. Click "Toggle device toolbar" (Ctrl+Shift+M)
4. Select "iPhone 12 Pro" or similar mobile device
5. Navigate to each page
6. Right-click > "Capture screenshot"
7. Save to this folder

### Option 2: Windows Snipping Tool

1. Open app in browser
2. Press Win+Shift+S
3. Select area to capture
4. Save to this folder

### Option 3: Third-party Tools

- ShareX (Free, powerful)
- Greenshot (Free)
- Lightshot (Free)

## Image Specifications

- **Format:** PNG (preferred) or JPG
- **Size:** Mobile view (375px - 428px width)
- **Quality:** High resolution
- **Content:** Show actual app screens, not mockups
